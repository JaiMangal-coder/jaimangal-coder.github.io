1.1 Altham’s Multiplicative Binomial Distribution

The PMF for Altham’s Multiplicative Binomial Distribution is \[f_X(x) = \frac{1}{c} {n \choose x} p^x (1-p)^{n-x} \theta^{x(n-x)}I(x \in \{0, 1, ..., n\})\] where \[c = \sum_{i=0}^n {n \choose i} p^i (1-p)^{n-i} \theta^{i(n-i)}.\] Just so we start on the same page, I have included my R function for computing probability using this distribution.

Code 
daltbinom <- function(x, size, prob, theta){
  normalizing.constant <- 0
  for(j in 0:size){
    normalizing.constant <- normalizing.constant + 
      choose(size, j) * prob^j * (1-prob)^(size-j) * theta^(j*(size-j))
  }
  # Probability Mass at x=x
  (1/normalizing.constant) *
    choose(size, x) * prob^x * (1-prob)^(size-x) * theta^(x*(size-x)) *
      (x>=0)*(x<=size)
}

2 Altham’s Multiplicative Beta Binomial Distribution

The PMF for Altham’s Multiplicative Beta Binomial Distribution is \[f_X(x) = \left[\int_{0}^1 \left(\frac{1}{c} {n \choose x} p^x (1-p)^{n-x} \theta^{x(n-x)}\right)\left(\frac{\Gamma(\alpha+\beta)}{\Gamma(\alpha)\Gamma(\beta)} p^{\alpha-1}(1-p)^{\beta-1}\right) dp\right]I(x \in \{0, 1, ..., n\})\]

Just so we start on the same page, I have included my R function for computing probability using this distribution.

Code 
daltbbinom.single <- function(x, size, alpha, beta, theta){
    integrand <- function(p) {
    sapply(p, # integrate will pass in a vector, so we use sapply to apply the following
           function(p_val){
             # f(x|p) * f(p|alpha, beta)
             daltbinom(x, size, p_val, theta) * dbeta(p_val, alpha, beta)
           }
    )
  }
  # Probability Mass at x=x
  # integrate(integrand, lower = 0, upper = 1)$value * (x>=0)*(x<=size)
  # I shrink the bounds below to avoid log(0)=-infinty
  # I use try() to catch when there is an error and avoid crashing
  res <- try(integrate(integrand, lower = 1e-7, upper = 1 - 1e-7)$value * 
                  (x>=0)*(x<=size), 
             silent = TRUE)
  # If there is an error, return a near-zero probability
  if(inherits(res, "try-error")) return(0.1^6)
  
  res
}
# Write a function that works on a vector
daltbbinom <- function(x, size, alpha, beta, theta){
  sapply(X=x, FUN=daltbbinom.single, size=size, alpha=alpha, beta=beta, theta=theta)
}

4.1 Part a

Altham’s Multiplicative Binomial Distribution does not typically simplify to the clean \(E(X)=np\) seen in the Binomial distribution, unless \(\theta = 1\). Write out the expression that would need to be solved to find \(E(X^k)\), and then write a function that computes it given the proposed parameter values \(p\) (prob) and \(\theta\) (theta).

Solution

To solve \(E(X^k)\), the expression follows:

\[E(X^k) = \sum_{x=0}^{n} x^k P(X=x). \]

Then, I just created a simple function that emulates this expression for varying prob and theta values, also size as well.

Code 
altham_moment = function(size, k, prob, theta){
  x_vals = 0:size

  moment = sum(sapply(x_vals, function(x){ #sums all together
    (x^k)*daltbinom(x = x, size = size, prob = prob, theta = theta)
    })
  )
  return(moment)
}

Solution

I made my function, remembering to keep the log inside the summation. It really is just the sum of the log probabilities. I also made my function support a size parameter because I’m cool like that.

Code 
altham_loglike = function(x, size, prob, theta){
  loglike = sum(log(
    sapply(x, function(x_val){
      daltbinom(x=x_val, size = size, prob = prob, theta = theta)
    })
  ))
  return(loglike)
}

ll_altham = function(par, data, size, mle = F){ #this was all adapted from class notes
  prob = ifelse(mle, 1 / (1 + exp(-par[1])), par[1]) #restricting
  theta = ifelse(mle, exp(par[2]), par[2])

  ll = altham_loglike(x = data, size = size, prob = prob, theta = theta)

  ifelse(mle, -ll, ll)
}

get_altham_mle = function(data, size){ #function to grab params by player
  mle = optim(
    par = c(0, 0), #starting val
    fn = ll_altham, #my function for optim
    data = data,
    size = size,
    mle = T
  )

  return(c(prob = 1 / (1 + exp(-mle$par[1])), theta = exp(mle$par[2])) #results here
  )
}

ll_altham_beta = function(par, data, size, mle = F){ #same step where i adapted from class notes
  alpha = ifelse(mle, exp(par[1]), par[1]) #restricted by making them all positive via exponents
  beta = ifelse(mle, exp(par[2]), par[2])
  theta = ifelse(mle, exp(par[3]), par[3])

  probs = daltbbinom(x = 0:6, size = size, alpha = alpha, beta = beta, theta = theta) #precomputed probs as given in the note 

  log_probs = log(probs) #now logged

  ll = sum(log_probs[data + 1]) #need data+1 
  #for indexing (stupid R indexes at 1 first, although this is nicer to read, i forgot abt this)

  ifelse(mle, -ll, ll)
}

get_altham_beta_mle = function(data, size){ #same idea as before

The Altham’s Multiplicative Beta-Binomial Model gave a more robust analysis by allowing the leg-win probability to vary a little bit across matches. The fitted beta means did well to predict standings overall here. Luke Littler had the highest with 0.425, followed by Luke Humphries, and again Peter Wright had the lowest at 0.197. The estimated theta values varied a lot here. Overall, this model gave a much more thorough analysis of the players’ qualities and matches’ variability.
